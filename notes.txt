REACT HOOKS 

> useState : to manage state 
> useEffect : to handle side effects (like API call, DOM manipulation, event listener)
> useContext : to consume global state with props drilling
> useReducer : for complex state management (like small version of Redux)
> useRef : to hold mutable values so that they don't trigger re-rendering, or to access DOM
> useMemo & useCallback : for optimization, to avoid unnecessary re-renders
